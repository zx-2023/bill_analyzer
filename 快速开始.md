## 🆘 常见问题

### Q1: 提示"python3: command not found"

**解决方案：**

检查Python是否已安装：
```bash
python --version
# 或
python3 --version
```

如果未安装，请访问 https://www.python.org/downloads/ 下载安装。

---

### Q2: 提示"Permission denied"

**解决方案：**

给脚本添加执行权限：
```bash
chmod +x setup_env.sh start.sh
```

---

### Q3: 依赖安装失败

**常见原因：**
1. 网络问题
2. Python版本不兼容

**解决方案：**

使用国内镜像源：
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

或指定Python版本：
```bash
python3.10 -m venv venv
```

---

### Q4: 如何更新依赖？

```bash
# 激活环境
source venv/bin/activate

# 更新所有包
pip install --upgrade -r requirements.txt
```

---

### Q5: 如何完全重置环境？

```bash
# 1. 删除虚拟环境
rm -rf venv

# 2. 重新运行配置脚本
bash setup_env.sh
```

## 💡 环境管理最佳实践

### 使用venv（推荐）
```bash
# 创建
python3 -m venv venv

# 激活（macOS/Linux）
source venv/bin/activate

# 激活（Windows）
venv\Scripts\activate

# 退出
deactivate
```

### 使用Conda（如果已安装）
```bash
# 创建
conda create -n bill_analyzer python=3.10

# 激活
conda activate bill_analyzer

# 退出
conda deactivate

# 删除
conda remove -n bill_analyzer --all
```
